📚 Sistema de Notas Acadêmicas em Java






📖 Sobre o projeto

Este projeto consiste em um sistema simples desenvolvido em Java para gerenciamento de notas acadêmicas. A aplicação permite cadastrar informações do aluno, inserir suas notas e calcular automaticamente a média final, informando sua situação acadêmica.

O sistema foi criado com foco em prática de lógica de programação e Programação Orientada a Objetos (POO).

🚀 Funcionalidades

✔ Cadastro do aluno
✔ Registro da matéria
✔ Registro do professor
✔ Inserção de notas
✔ Cálculo automático da média
✔ Exibição do resultado final do aluno

🧠 Regras de negócio

O sistema avalia o aluno com base na média das notas:

✅ Aprovado → Média ≥ 7
⚠️ Recuperação → Média entre 5 e 6.9
❌ Reprovado → Média < 5
